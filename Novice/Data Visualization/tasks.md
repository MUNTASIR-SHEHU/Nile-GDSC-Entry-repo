# Data Visualization Tasks

## Task 1: Interactive Dashboard
Create an interactive dashboard with multiple linked charts.

**Requirements:**
- Load a multi-dimensional dataset
- Create at least 4 different chart types
- Implement interactivity (filters, hover effects)
- Ensure responsive design
- Add clear labels and legends

## Task 2: Geospatial Visualization
Build a map-based visualization with data overlays.

**Requirements:**
- Load geographic data
- Create a choropleth or heatmap
- Add data points or markers
- Implement zoom and pan functionality
- Create a color scale that shows patterns clearly

## Task 3: Time Series Animation
Create an animated visualization showing data changes over time.

**Requirements:**
- Load time series data
- Create animation showing temporal changes
- Include play/pause controls
- Add timeline scrubber
- Highlight key trends or anomalies
