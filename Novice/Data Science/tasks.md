# Data Science Tasks

## Task 1: Complete Data Pipeline
Build an end-to-end data pipeline from raw data to insights.

**Requirements:**
- Load messy real-world dataset (with missing values, outliers)
- Clean and preprocess the data
- Perform exploratory data analysis
- Extract 5 meaningful insights
- Create a summary report

## Task 2: Predictive Model with Feature Engineering
Build a predictive model with custom feature engineering.

**Requirements:**
- Load a dataset for prediction
- Create at least 3 new engineered features
- Handle imbalanced data if present
- Build and train a model
- Achieve reasonable accuracy (>75%)
- Explain feature importance

## Task 3: A/B Testing Analysis
Analyze A/B test results and make recommendations.

**Requirements:**
- Generate or load A/B test data
- Perform statistical significance testing
- Calculate confidence intervals
- Visualize test results
- Make data-driven recommendations
