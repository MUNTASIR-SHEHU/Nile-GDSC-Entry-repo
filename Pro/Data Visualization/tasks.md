# Data Visualization Tasks

## Task 1: Custom 3D Interactive Scientific Visualization
Build a publication-quality 3D visualization system for complex scientific data.

**Requirements:**
- Visualize volumetric or multi-dimensional scientific data
- Implement custom WebGL/Three.js rendering pipeline
- Add ray tracing or volume rendering capabilities
- Create interactive slicing planes and iso-surfaces
- Implement VR/AR viewing mode
- Handle datasets with millions of data points
- Optimize for 60fps on standard hardware
- Export high-resolution images and videos

## Task 2: Real-time Collaborative Data Exploration Platform
Build a multi-user collaborative visualization platform.

**Requirements:**
- Implement WebSocket-based real-time synchronization
- Support simultaneous users (10+) interacting with same data
- Add collaborative annotations and comments
- Implement user-specific views with shared insights
- Create version control for visualization states
- Add chat/communication layer
- Handle conflict resolution when users modify same elements
- Scale to datasets with 1M+ rows

## Task 3: Automated Insight Discovery and Visualization
Create AI-powered system that automatically finds and visualizes insights.

**Requirements:**
- Implement automated exploratory data analysis
- Use ML to detect patterns, anomalies, and trends
- Auto-generate appropriate chart types for discovered insights
- Create natural language descriptions of findings
- Rank insights by statistical significance
- Build interactive drill-down capabilities
- Generate executive summary dashboard automatically
- Handle diverse data types (numerical, categorical, temporal, spatial)
