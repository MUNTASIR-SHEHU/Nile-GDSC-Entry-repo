# Data Science - Intermediate Level

Welcome to the Data Science section for intermediate level exercises.

## Overview

This folder contains advanced data science exercises requiring strong analytical and technical skills.

## Getting Started

Open the `starter.ipynb` notebook to begin working on the exercises.

## Requirements

- Python 3.x
- Jupyter Notebook
- numpy

## Instructions

1. Open `starter.ipynb` in Jupyter Notebook or JupyterLab
2. Review the three tasks available in the notebook
3. **Complete ANY TWO tasks**
4. Refer to `tasks.md` for detailed requirements for each task

Good luck!
