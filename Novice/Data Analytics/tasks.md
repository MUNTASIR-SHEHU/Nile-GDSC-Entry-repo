# Data Analytics Tasks

## Task 1: Customer Segmentation Analysis
Segment customers based on behavior and demographics.

**Requirements:**
- Load customer dataset (purchase history, demographics)
- Perform RFM analysis (Recency, Frequency, Monetary)
- Create customer segments using clustering
- Profile each segment with key characteristics
- Provide actionable business recommendations

## Task 2: Sales Funnel Optimization
Analyze conversion funnel and identify drop-off points.

**Requirements:**
- Load funnel data (multiple stages)
- Calculate conversion rates at each stage
- Identify bottlenecks and drop-off points
- Perform cohort analysis
- Suggest specific optimization strategies

## Task 3: Churn Prediction and Analysis
Build a model to predict customer churn.

**Requirements:**
- Load customer data with churn labels
- Identify key churn indicators
- Build predictive model (>80% accuracy)
- Calculate customer lifetime value
- Create retention strategy recommendations
