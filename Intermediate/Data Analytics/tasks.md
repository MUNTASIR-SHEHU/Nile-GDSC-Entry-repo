# Data Analytics Tasks

## Task 1: Advanced Cohort Analysis with Predictive Insights
Perform deep cohort analysis and predict future behavior.

**Requirements:**
- Load multi-month transactional data
- Create cohort retention matrices
- Calculate cohort-specific metrics (LTV, churn rate, ARPU)
- Identify cohort characteristics that predict long-term value
- Build model to predict which cohort new users will belong to
- Visualize cohort evolution over time
- Provide specific recommendations for each cohort

## Task 2: Market Basket Analysis and Recommendation System
Analyze purchase patterns and build recommendation engine.

**Requirements:**
- Load transaction data with product information
- Implement Apriori or FP-Growth algorithm
- Find association rules with statistical significance
- Calculate lift, confidence, and support metrics
- Build collaborative filtering recommendation system
- Evaluate recommendations using precision@k and recall@k
- Create business rules based on discovered patterns

## Task 3: Multi-Touch Attribution Modeling
Determine credit allocation across marketing channels.

**Requirements:**
- Load customer journey data (multiple touchpoints)
- Implement multiple attribution models (first-touch, last-touch, linear, time-decay)
- Build data-driven attribution using Markov chains or Shapley values
- Compare models and analyze differences
- Calculate ROI by channel
- Perform sensitivity analysis
- Recommend budget reallocation strategy
